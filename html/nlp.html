Natural language processing with a multi-layer perseptron

This is the neural network for sentiment analysis described in Andrew Trask's excellent book "Grokking Deep Learning". The aim is for the network to use the text and the star rating of a collection of film reviews downloaded from the udacity deep-learning mooc. The data can be retrieved using the following commands. 

    
wget https://github.com/udacity/deep-learning/raw/master/sentiment-network/labels.txt
wget https://github.com/udacity/deep-learning/raw/master/sentiment-network/reviews.txt
